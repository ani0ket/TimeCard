# Employee Shift Analysis

This repository contains a Node.js script for analyzing employee shift data. The script reads data from an Excel file and performs various analyses, such as identifying employees who have worked for 7 consecutive days, those with less than 10 hours between shifts (but more than 1 hour), and those who have worked for more than 14 hours in a single shift.
